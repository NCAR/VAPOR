Natively Supported Data Formats
_______________________________

Some data formats can be easily loaded into Vapor through the File->Import menu.

.. figure:: /_images/importData.png
    :width: 400
    :align: center
    :figclass: align-center

    Importing a data set into Vapor.

Vapor's "natively supported" data formats are:

.. toctree::
   :maxdepth: 1 

   wrf
   mpas
   NetCDF (CF-Compliant) <netCDF/cfComplianceBrief>
   brickOfValues
   dcp

.. note::

    NetCDF (CF-Compliant) and Brick of Values (BOV) data come with caveats that may require you to provide additional data to Vapor.  Read more under their respective documentation.
