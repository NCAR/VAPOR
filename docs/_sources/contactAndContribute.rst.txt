.. _contactAndContribute:

======================
Contact and Contribute
======================

.. _getHelp:

`Get Help <https://github.com/NCAR/VAPOR-GitHubPage/issues>`_
_____________________________________________________________

If something is not working for you, you can submit an issue at the GitHub repository `here <https://github.com/NCAR/VAPOR-GitHubPage/issues>`_.

Note that this is not Vapor's code repository, which we use to track bugs and enhancement requests.  See below on how to do that.

.. _reportABug:

`Report A Bug <https://github.com/NCAR/VAPOR/issues/new/choose>`_
_________________________________________________________________

If you believe you have found a bug within Vapor, please report the issue at this `GitHub repository <https://github.com/NCAR/VAPOR/issues/new/choose>`_.

Choose "Bug Report", and please fill out all fields in the issue template as completely as possible.

.. _submitAnEnhancement:

`Request An Enhancement <https://github.com/NCAR/VAPOR/issues/new/choose>`_
___________________________________________________________________________

If there is an enhancement or feature that you think would help your science, feel free to suggest it to our development team at the following link to our `GitHub repository <https://github.com/NCAR/VAPOR/issues/new/choose>`_.

Choose "Enhancement Request", and please fill out all fields in the issue template as completely as possible.

.. _forkOnGithub:

`Fork Vapor on GitHub <https://github.com/NCAR/VAPOR>`_
_______________________________________________________

Anyone can clone `Vapor's repository <https://github.com/NCAR/VAPOR>`_ and submit a pull request to add their own contributions

Please review our `coding conventions <https://github.com/NCAR/VAPOR/wiki/Vapor-Coding-Convention>`_ before submitting a pull request.
